# Changelog for Berliner Allerlei Vocabulary

## Development

## [0.0.3](https://github.com/berlin/lod-vocabulary/blob/blnal_0.0.3/data/blnal/blnal.ttl)

_(2026-01-13)_

- Add the [E-Government-Gesetz Berlin](https://gesetze.berlin.de/perma?d=jlr-EGovGBErahmen) as an [eli:LegalResource](http://data.europa.eu/eli/ontology#LegalResource) to `blnal`. This is a temporary addition to have a resource identifier to use as the [dcatap:applicableLegislation](http://data.europa.eu/r5r/applicableLegislation) in Berlin's open data portal's DCAT API. If there is ever official metadata for this law, deprecate it here and replace.
